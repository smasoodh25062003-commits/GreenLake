How It Works

Get started in 3 simple steps

Follow the guide below to retrieve your authentication credentials from the GreenLake portal.

step1.png
Start Here
Launch the HPE GreenLake CCS Support Portal
Open your browser and navigate to the HPE GreenLake CCS Support Portal. Make sure you are fully logged in before proceeding to the next step.
step2.png
DevTools
Open Inspect Element & Go to the Network Tab
Press F12 (or right-click → Inspect) to open DevTools. Click the Network tab, then type any random characters in the portal's search bar — this triggers a network request you need to capture.
03
step3.png
Copy Headers
Find & Copy Your Authorization and Cookie
In the Network tab, look for a request named customers?limit=10&offset (Customers section) or devices?limit… (Devices section). Click it → open the Headers tab → find Authorization and Cookie — copy both values in full.
